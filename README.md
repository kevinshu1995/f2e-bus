# The F2E 全台公車動態時刻查詢應用服務

## 專案運行

-   npm

```bash
npm install
// Develop
npm run dev

// Build
npm run build
```

---

## 相關技術/插件

1. Vite
2. Tailwind
3. Vue 3
    1. Vue router
4. leaflet
5. Axios
